## Faithful Maggiar & Sadighian (2017) model

The faithful pricing-coupled model (`joint_pricing_removal_env.rs` + `price_dependent_gamma_demand.rs`
+ `joint_pricing_removal_dp.rs`) is verified by RE-RUNNING the env and finite-horizon DP and asserting
it reproduces what the paper actually states:

- the log-linear price/demand map and the `E = -beta*p0` elasticity relation,
- the Gamma(mean mu_t, CV=1) noise quantile discretization,
- the Eq. 4 single-period reward, computed by hand, including the `h- = c + k` backorder convention and
  the `s*min(x,y) + l*max(x-y,0)` terminal,
- **the paper's PROVEN optimal-policy structure (Lemma 3.1, Section 3.2 / 7.2.1)**:
  - markdown / target demand nondecreasing in inventory, nonincreasing in returnable level,
  - returns nondecreasing in inventory,
  - purchases nonincreasing in inventory,
  - value-function supermodularity (L♮-concavity of `V_t`),
- the Table-1 NPV-surface MAGNITUDE: the env-computed peak at `t=24` brackets the reported `~84000` top
  contour (Figure 7).

What is NOT exactly reproduced: the per-period `mu_t` profile is given only graphically (Figure 6), and
the headline output is a NPV SURFACE at fixed `t=24`, not a tabulated cost. The exact `~84000` is
therefore not reproducible to tight tolerance, so `literature_verified` stays `false` for the NPV
figure. The structural-property reproduction is exact and is the literature-grounded correctness anchor.
