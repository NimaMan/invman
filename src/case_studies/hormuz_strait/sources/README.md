# sources

Source tracking for the Strait of Hormuz model.

Files:

- `source_manifest.csv`
  - canonical list of URLs, publication dates, access date, local filenames, and notes
- `checksums.sha256`
  - integrity hashes for the checked-in raw files

This folder is part of the reproducibility contract for the problem.

Current tracked source groups:

- 2024 Hormuz flow and destination data from EIA
- the EIA daily prices page snapshot used for the `2026-04-06` analysis
- the March `2026` EIA STEO PDF used for the month-ahead Brent anchors
- the OPEC+ `2026-04-05` release used for the May `2026` supply-adjustment anchor
