# Docs

This directory contains project documentation that belongs with the `invman` repo.

Structure:

- `benchmarks/`: benchmark notes, refresh logs, and exported benchmark summaries
- `literature/`: literature notes and local reference material used while building the benchmark sets
- `architecture/`: architecture diagrams and project-structure references
- `rust/`: Rust crate guide and verification conventions

Important benchmark notes:

- `benchmarks/lost_sales_l4_refresh/README.md`
- `benchmarks/fixed_cost_l4_refresh/README.md`
- `benchmarks/fixed_order_cost_literature_subset_poisson_mu5/README.md`

Important literature notes:

- `literature/fixed_order_cost_literature/README.md`
- `literature/fundamental_problem_families/README.md`

Problem-family-specific literature notes should live in the associated problem README under
`src/problems/<problem>/README.md`.
